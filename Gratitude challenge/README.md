# 💟 30-Day Gratitude Challenge

Welcome to my **30-Day Gratitude Challenge**! 

This is a beginner-friendly web project I built to encourage gratitude, self-reflection, and journaling. The website contains 30 unique prompts—one for each day—to inspire positive thinking and help users build a daily journaling habit.

---

##  Features

*  30 unique gratitude and self-reflection prompts
*  A dedicated writing page for each day
*  Soft pastel-themed design
*  Simple and clean user interface

---

## 🛠️ Built With

* HTML5
* CSS3

---

## 📌 Project Note

For the `write.html` page, I used HTML and CSS together in the same file by placing the CSS inside a `<style>` section. I chose this approach to keep the page simple while learning. As I continue improving my skills, I plan to separate the styles into their own CSS file.
